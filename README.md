# TripMind

AI-powered travel planning platform that generates personalized itineraries based on destination, budget, travel style, pace, interests, and food preferences.

## Live Demo

https://tripmind-phi.vercel.app

## Features

- AI-generated travel itineraries
- Stay recommendations
- Restaurant recommendations
- Day-wise sightseeing plans
- Packing lists
- Budget analysis and charts
- AI travel insights
- AI chat assistant
- PDF export

## Tech Stack

### Frontend
- React
- Recharts
- HTML/CSS

### Backend
- Flask
- Python
- Groq API

### Deployment
- Vercel
- Render
